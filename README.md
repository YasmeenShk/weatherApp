# WeatherGlass 🌤️

A modern **glassmorphism weather web app** that shows real-time weather

## 🔗 Live Demo
[WEATHERAPP LIVE LINK](https://weatherglass.netlify.app)

## 📸 Preview

A sleek weather dashboard with animated background, quick city search, and detailed weather stats.

## ✨ Features

* 🔎 Search weather by city name
* 🌡️ Current temperature and weather condition
* 💧 Humidity information with description
* 🌬️ Wind speed and direction
* 👁️ Visibility status
* 🧭 Atmospheric pressure
* ☀️ UV Index indicator
* 🌅 Sunrise and Sunset time
* 📅 7-Day weather forecast
* ⭐ Animated star background
* 🏙️ Quick city buttons for fast search

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **JavaScript (Vanilla JS)**
* **Open-Meteo Weather API**

## ⚙️ How to Run Locally

1. Clone the repository

```
git clone https://github.com/YasmeenShk/weatherApp.git
```

2. Open the project folder

```
cd weatherApp
```

3. Run Tailwind build

```
npm run dev
```

4. Open `index.html` in your browser.

## 🌍 API Used

Weather data is fetched from the **Open-Meteo API**.
